--------------------------------------------------------------------------------
STEP-BY-STEP IMPLEMENTATION
--------------------------------------------------------------------------------

STEP 1: AWS CONSOLE LOGIN
- Log into AWS Console.
- Ensure you are in the eu-west-2 Region.

STEP 2: CREATE EC2 INSTANCE (PASSES TC1)
- Navigate to EC2 -> "Launch Instance".
- Name: `<your-labskraft-username>-<your-exam-code>` (e.g. `labs-kraft-demo106-labskraft-team`).
- Select Windows Server AMI.
- Select t2.micro Instance Type.

STEP 3: CREATE EBS VOLUME (PASSES TC2)
- Go to Volumes -> "Create volume".
- Size: 20 GB, Type: io2, IOPS: 500.

STEP 4: ATTACH & MOUNT EBS VOLUME (PASSES TC3)
- Attach the volume to the instance.
- RDP into the instance.
- Open Disk Management, initialize disk.
- Format as NTFS, assign drive letter F:.
