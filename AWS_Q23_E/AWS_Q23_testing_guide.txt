--------------------------------------------------------------------------------
STEP-BY-STEP IMPLEMENTATION
--------------------------------------------------------------------------------

STEP 1: AWS CONSOLE LOGIN
- Log into the AWS Management Console using your assigned student credentials.
- Ensure you are operating in the eu-west-2 Region.

STEP 2: CREATE EC2 INSTANCE (PASSES TC1)
- Navigate to the EC2 Dashboard -> Click "Launch Instance".
- Enter the exact Instance Name: `labskraft-ubuntu-ec2-<your-labskraft-username>`.
- Select Ubuntu Server 22.04 LTS AMI.
- Select t2.micro Instance Type.

STEP 3: CREATE & ATTACH EBS VOLUME (PASSES TC2)
- Go to Volumes -> "Create volume".
- Size: 10 GB, Type: gp3.
- Attach the volume to the instance created in Step 2.

STEP 4: MOUNT EBS VOLUME (PASSES TC3)
- SSH into the EC2 instance.
- Format the volume with ext4 using `mkfs.ext4`.
- Mount it to `/mnt/data-store`.
