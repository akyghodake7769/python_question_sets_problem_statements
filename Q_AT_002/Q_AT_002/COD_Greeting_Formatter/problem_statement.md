# Greeting Formatter (Basic)

### Task
Implement the `formatGreeting` function in the `GreetingFormatter.kt` file.

### Requirements
The function should take a `String` called `name` and return a `String` in the following format:
`"Hello, [name]! Welcome to Kotlin."`

### Examples
- **Input**: `"Alice"` -> **Return**: `"Hello, Alice! Welcome to Kotlin."`
- **Input**: `"Bob"` -> **Return**: `"Hello, Bob! Welcome to Kotlin."`

### Constraints
- The `name` parameter will not be empty.
- Case sensitivity matters; ensure the exclamation mark and spacing match the requirements exactly.
